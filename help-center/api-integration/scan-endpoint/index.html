the scan endpoint

learn how to interact with the /v1/score endpoint to assess wallet risk.

endpoint overview

the /v1/score endpoint accepts a POST request with an ethereum wallet address. only ethereum addresses are currently supported.

POST https://api.sentinelpay.org/v1/score

request parameters

  • wallet (string) — the ethereum wallet address to scan (0x-prefixed, 42 characters).
